### 🏠 [Homepage](https://www.nicolasracchi.com)

## Install & Develop

```sh
# clone this repo
git clone https://github.com/nicolas-racchi/personal-website.git

# cd into project
cd personal-website

# install dependencies and run development version
npm install && npm run dev
```
